02 — Selected work

Energy-production optimisation at Electio

Machine Learning Engineer · 2026 – present

Physics-based models of complex energy-production systems turned into predictive, decision-support tools in production code. I developed a nodal-analysis and multiphase-flow model for production optimisation and integrated its predictions as physically-grounded features into a machine-learning model. I own the optimisation recommendations delivered to customers (e.g. gas-injection strategy), derived from historical operating data and model output.

Built in Python and shipped to production: Databricks and Spark for large-scale data, MLflow for the machine-learning lifecycle, InfluxDB for time-series operating data, and AWS with Docker for deployment.

Python · Databricks · Spark · MLflow · InfluxDB · AWS · Docker

giwaxs-analysis

Source ↗

Author · Open-source Python package

Python tools for GIWAXS data processing built on pyFAI and fabio: calibration, 1D/sector/cake integration, in situ heatmaps, peak fitting, and Scherrer crystallite-size analysis. Properly engineered and maintained — packaged, tested, documented, MIT-licensed, with a CITATION.cff for citable reuse.

In use by groups working on in situ GIWAXS, including the Riede group (Oxford), the Heutz group (Imperial), the University of Sheffield, the ESRF, and the Su Materials Lab (Lawrence Berkeley National Laboratory).

Python · pyFAI · fabio · X-ray scattering

giwaxs_toolbox

Source ↗

Co-author · Diamond Light Source · beamline I07

Python tools for visualising and analysing GIWAXS data collected on Diamond Light Source beamline I07, with interactive widgets for exploring in situ datasets and extracting 1D profiles. A maintained, facility-grade package: src-layout, unit tests, GitHub Actions CI, Sphinx documentation with training videos, and containerised (Docker) deployment on the Diamond filesystem for beamline users.

03 — Personal projects

pollock

Source ↗

Personal project · Creative coding · Python

A creative-coding experiment at the meeting point of art and science. Inspired by Richard Taylor's work on the fractal signature of Jackson Pollock's drip paintings, I built a configurable pendulum simulation — choose the number of arms, the colours, the starting points and the random seed — that scatters its trajectories across a canvas to emulate a Pollock, then measures the fractal dimension of the result.

With several arms swinging under gravity, the motion is deeply chaotic: nudge a starting angle by a hair and the path diverges completely, so no two runs paint the same picture — hence the name.

Python · NumPy · matplotlib

05 — Recognition & impact

Research infrastructure

An X-ray method awarded standing beamtime at the UK's national synchrotron

The in situ X-ray scattering technique I led was awarded Block Allocation Group (BAG) status at Diamond Light Source — a guaranteed recurring block of beamtime (roughly two weeks every operating cycle, over several years), granted only to methods a facility judges mature and broadly valuable enough for the wider research community to use. As lead investigator and BAG superuser I ran the MINERVA vacuum chamber and the scattering experiment simultaneously, trained incoming users, and authored the software and standard operating procedures that opened the technique to other teams — now used by groups at Imperial, TU Dresden, Sheffield and Florence.
